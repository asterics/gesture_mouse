# Instructions for Chrome Dino Game with Jaw Gesture

Play the chrome dino game with two gestures:
* ```Brow Up```: Press and release <kbd>SPACE</kbd>
* ```Jaw Open```: Hold as long as open key <kbd>Cursor Down</kbd>

## Init gesture_mouse

1. Open ```gesture_mouse``` folder using PyCharm:
Select folder in explorer and right click on ```Open folder as PyCharm Community Edition Project```
2. Run gesture_mouse.py by entering the following command into the terminal window:
```poetry run python gui.py```
3. Go to tab ```Keyboard```, click on ```Load profile``` and open file ```config/profiles/chrome-dino-broweinnerup-jawopen.json```
4. Open the browser chrome and enter ```chrome://dino``` into the address bar.
5. Toggle start/stop of gesture tracking by the shortcuts: <kbd>Ctrl</kbd><kbd>Alt</kbd>+<kbd>v</kbd>
